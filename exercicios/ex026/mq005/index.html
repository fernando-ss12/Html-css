Ao clicar no menu hamburgher, o menu deverá ser fechado e os itens serão exibidos em uma lista horizontal. Use o CSS Flexbox para este exercício.

Adicione um botão "Fechar Menu" para fechar o menu em dispositivos móveis. Use o CSS Grid para organizar os itens do menu em uma grade 3x3.
